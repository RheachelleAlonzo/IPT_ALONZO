Project Overview

"This project was assigned by our professor as part of a school assignment to create a portfolio website. The portfolio showcases our skills and accomplishments, including a quiz application that allows users to take quizzes, view results, and track progress. The project is developed using HTML, CSS, and JavaScript for the front-end, with data stored either locally or in a backend database. The development process involved using GitHub for version control and collaboration, and Visual Studio Code (VSCode) as the code editor."

Tools & Technologies

  • HTML: Used to structure the content of the webpage, including forms for quiz questions, results, and user input.
  • CSS: Used for styling the page to make it visually appealing and responsive, including animations and hover effects.
  • JavaScript: Handles quiz logic, question navigation, score tracking, and dynamic content changes.
  • GitHub: The project is version-controlled using Git and hosted on GitHub for collaboration and sharing.
  • Visual Studio Code: The code editor used for writing, editing, and debugging the code.

Project Structure

The project follows a simple structure:

  • home.html: The main HTML page that contains the structure for the homepage.
  • quizzes.html: Contains the quiz interface and logic for displaying questions and processing results.
  • styles.css: Contains the styling rules for the website, including layout, fonts, and animations.
  • scripts.js: Contains JavaScript functions for quiz functionality, such as scoring and question navigation.

Future Improvements

In the future, I plan to enhance the project by adding the following features:

  • Backend integration for storing user data and quiz results.
  • Difficulty levels for the quizzes (easy, medium, hard).
  • More interactive and engaging UI features.
